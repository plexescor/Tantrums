What Is Tantrums?

Tantrums compiles .42AHH source files into .42ass bytecode, which runs on a custom stack-based virtual machine. It has optional static typing, a 7-layer memory safety system built without a garbage collector or borrow checker, and built-in profiling with zero imports required.

Yes, those are the real file extensions. No, they are not changing.

The Stories Behind The Language

The documentation is honest about everything that went wrong. These are real disasters that shaped the architecture.

ğŸ”¥ The 32GB RAM Incident
Why the garbage collector doesn't exist

A self-referencing map inside while(true) created reference cycles the original GC could never break. 16GB of RAM consumed. Then 16GB of page file. Hard reboot. The GC was removed on day 2 and replaced with the 7-layer system you see today.
