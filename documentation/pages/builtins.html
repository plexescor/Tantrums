input() Type Casting

tantrums (.42AHH)
x = input("Enter:");         // → string (default)
string s = input("Name:");  // → string
int n = input("Number:");  // → parsed as integer
float f = input("Float:");  // → parsed as float
bool b = input("Yes/No:"); // "true"→true, "false"→false,
                               // empty/whitespace→false, anything else→true

range()

tantrums (.42AHH)
range(10)           // 0, 1, 2, ..., 9
range(2, 10)        // 2, 3, 4, ..., 9
range(0, 50, 5)     // 0, 5, 10, 15, ..., 45
✓
Zero Memory

range() uses an internal ObjRange type. No matter how large the range, it allocates a single counter — not a list of N items. range(100000000) uses the same memory as range(1).
