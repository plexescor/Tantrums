ⓘ
Auto String Concat

When one side of + is a string, the other side is automatically converted. No explicit cast needed. "val: " + 42 → "val: 42". This only activates when one side IS a string — two non-strings won't silently coerce.

Fat-Finger Operators

=> and =< are intentional aliases. If you type them instead of >= / <=, the language understands. No silent error, no wrong result.

Division by Zero

⚠
Protected

Literal x / 0 is caught at compile time. Variable division like x / y where y evaluates to zero is caught at runtime — a clean error instead of a crash or platform-specific undefined behavior.
