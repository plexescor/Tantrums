Building from Source

Requirements, build commands, and verification.

Requirements

ToolVersionNotes
CMake3.15+Build system
MSVCVS 2022Windows (recommended)
GCC13+Linux / Windows (MinGW)
Clang16+Any platform
⚠
C++23 Required

The compiler uses C++23 features. Older compilers will fail. GCC 12, MSVC 2019, Clang 14 are NOT sufficient.

Build Commands

terminal
# Release build (always use for benchmarking or real use)
cmake -B build -S . -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release

# Debug build (for compiler development — ~2× slower)
cmake -B build -S . -DCMAKE_BUILD_TYPE=Debug
cmake --build build --config Debug

Verify Build

terminal
# Windows
.\build\Release\tantrums.exe run yourfile.42AHH

# Linux / Mac
./build/tantrums run yourfile.42AHH
