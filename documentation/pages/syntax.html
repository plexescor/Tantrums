Variables

Variables can be dynamic (untyped) or statically typed. In #mode both (the default), both styles coexist.

tantrums (.42AHH)
// Dynamic — no type annotation
x = 42;
name = "Tantrums";
alive = true;

// Typed declarations
int count = 0;
float pi = 3.14159;
string lang = "Tantrums";
bool ok = true;
list nums = [1, 2, 3];
map data = {{"key": "value"}};

// Default-initialized typed declarations
list items;   // → []
map kv;       // → {{}}
int y;        // → null
ⓘ
No Garbage Values

Typed declarations always initialize to a safe default. You will never read garbage memory from an uninitialized Tantrums variable.

Strings

Strings are double-quoted. Escape sequences are supported. Strings auto-concatenate with any other type when combined with +.

tantrums (.42AHH)
string s = "Hello, World!";
print(s);

// Auto string concat — no explicit cast needed
int age = 17;
print("Age: " + age);   // → Age: 17
print("Pi: " + 3.14);    // → Pi: 3.14

// Escape sequences
print("Line 1\nLine 2");
print("Tab:\there");
print("Quote: \"hello\"");
EscapeMeaning
\nNewline
\tTab
\\Backslash
\"Double quote
\rCarriage return
\0Null character

Lists

tantrums (.42AHH)
list nums = [1, 2, 3, 4, 5];

// indexing
print(nums[0]);          // → 1

// length
print(len(nums));        // → 5

// append
append(nums, 99);
print(nums[5]);          // → 99

// iterate
for item in nums {{ print(item); }}

Maps

Map keys must all be the same type. Mixing key types in one map is a runtime error.

tantrums (.42AHH)
map person = {{"name": "Alice", "age": "21"}};

// access
print(person["name"]);    // → Alice

// assign
person["city"] = "Jaipur";

// length
print(len(person));       // → 3
