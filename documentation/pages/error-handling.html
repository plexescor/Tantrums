Error Handling

throw, try/catch, and nested rethrow.

throw

Throw a string to halt execution with an error message:

tantrums (.42AHH)
throw "Something went wrong!";

try / catch

tantrums (.42AHH)
try
{{
    throw "oops";
}}
catch (e)
{{
    print("Caught: " + e);   // → Caught: oops
}}

// Without error variable
try {{ throw "fail"; }}
catch {{ print("Something failed"); }}

Nested try / catch with rethrow

tantrums (.42AHH)
try
{{
    try
    {{
        throw "inner error";
    }}
    catch (e)
    {{
        print("Inner caught: " + e);
        throw "re-thrown from inner";
    }}
}}
catch (e)
{{
    print("Outer caught: " + e);
}}
ⓘ
Throw Is Always A String

Tantrums throws string messages, not exception objects. No exception hierarchy to learn. No checked vs unchecked. No imports needed. The error IS the message.
