Installation

Tantrums is built from source using CMake. There are no pre-built binaries yet — you compile the compiler.

⚠
Requirements

CMake 3.15+, and a C++23 capable compiler: MSVC (Visual Studio 2022), GCC 13+, or Clang 16+.

terminal
# Clone the repo
git clone https://github.com/plexescor/Tantrums.git
cd Tantrums

# Configure (Release is recommended — debug build is ~2× slower)
cmake -B build -S . -DCMAKE_BUILD_TYPE=Release

# Build
cmake --build build --config Release

The resulting binary is at build/Release/tantrums.exe (Windows) or build/tantrums (Linux/Mac).

Hello World

Create a file named hello.42AHH:

tantrums (.42AHH)
tantrum void main()
{{
    print("Hello, World!");
}}
ⓘ
Entry Point

The main() function is called automatically. It must be named exactly main. No return type is required for main.

Running Files

terminal
# Compile + execute in one step
tantrums run hello.42AHH

# Output:
# Hello, World!

Compiling vs Running

You can separate compilation and execution. The compiled .42ass bytecode is cross-platform — compile on Windows, run on Linux.

terminal
# Step 1: compile to bytecode
tantrums compile hello.42AHH
# → produces hello.42ass in the same directory

# Step 2: execute the bytecode
tantrums exec hello.42ass

# The .42ass file can be moved to any OS
# as long as the Tantrums VM binary is present for that OS
