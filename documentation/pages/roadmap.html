Roadmap

What's planned, what's not planned, and why.

Tantrums is in active development. The roadmap below reflects the developer's stated goals. Nothing here is guaranteed — this is what the developer wants to build.

ⓘ
Development Context

The developer is a student building this during school holidays and exam prep windows. Development velocity is high during holiday blocks and slower during exam season. This roadmap reflects medium-to-long term ambitions.

⚠
Native Compilation Reality

The developer has been honest: native compilation will be complex even with AI assistance. The most achievable near-term path is C transpilation — generate C from .42AHH source, then compile with GCC/Clang. True LLVM backend JIT is a longer-term goal.

Intentionally Not Planned

FeatureReason
Pointer arithmeticVM pointers are registry indices, not real addresses. Implemented, tested, produced garbage. Removed.
Garbage collectorCaused 32GB RAM consumption on day 2. Replaced with 7-layer system. Not coming back.
Implicit truthy/falsyIntentional design choice. if() requires bool. This is a feature, not a gap.
