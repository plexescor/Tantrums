Control Flow

if/else, while, for-in, break, continue.

If / Else

tantrums (.42AHH)
if (x > 10)
{{
    print("big");
}}
else if (x == 10)
{{
    print("ten");
}}
else
{{
    print("small");
}}
⚠
No Implicit Truthy

The condition in if() must be a bool. if(myList) will throw a runtime error. Use if(len(myList) > 0).

While Loop

tantrums (.42AHH)
int count = 5;
while (count > 0)
{{
    print(count);
    count--;
}}

For-In Loop

Three variants: range, list iteration, string character iteration.

tantrums (.42AHH)
// range(end) — 0 to end-1
for i in range(10) {{ print(i); }}

// range(start, end)
for i in range(2, 10) {{ print(i); }}

// range(start, end, step)
for i in range(0, 50, 5) {{ print(i); }}

// iterate list
for item in [1, 2, 3] {{ print(item); }}

// iterate string characters
for ch in "Hello" {{ print(ch); }}
ⓘ
ObjRange Optimization

range(N) does NOT allocate a list of N items. It uses an internal ObjRange type — a counter with a limit. Memory usage is O(1) regardless of how large N is. This is why range(100_000_000) runs 11× faster than Python.

Break & Continue

tantrums (.42AHH)
for i in range(10)
{{
    if (i == 5) {{ break; }}
    if (i % 2 == 0) {{ continue; }}
    print(i);  // prints: 1 3
}}
