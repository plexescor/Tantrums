================================================================================
               TANTRUMS VS CODE EXTENSION — GUIDE
================================================================================

Version  : 0.2.0
Extension folder: tantrums-vscode/


================================================================================
INSTALLATION
================================================================================

  The extension is auto-installed to:
    %USERPROFILE%\.vscode\extensions\tantrums-dev.tantrums-lang-0.2.0\

  To manually install:
    1. Copy the tantrums-vscode/ folder to the path above
    2. Restart VS Code
    3. Open any .42AHH file — the extension activates automatically

  For file icons:
    File > Preferences > File Icon Theme > "Tantrums File Icons"
